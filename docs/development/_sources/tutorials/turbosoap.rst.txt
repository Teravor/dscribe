TurboSOAP - a fast implementation of SOAP
=========================================
TurboSOAP is an implementation of the SOAP scheme which enables more control 
over the various aspects of the descriptor

TurboSOAP is not directly maintained by the DScribe core team and requires
installation of separate python package turbosoap-dscribe. For information
on installation and examples please see 
`TurboSOAP documentation <https://mcaroba.github.io/turbosoap_dscribe/>`_
